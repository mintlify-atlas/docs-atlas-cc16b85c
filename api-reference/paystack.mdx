The `Paystack` class provides a wrapper for the Paystack API in Laravel Cashier Paystack.

## Core API Method

### `api()`

Perform a Paystack API call.

```php
public static function api(string $uri, string $method, array $payload = []): Response
```

**Parameters:**

- `$uri` (string): The API endpoint URI
- `$method` (string): The HTTP method (get, post, put, delete)
- `$payload` (array): The request payload

**Throws:** 
- `Exception` if API key is not set
- `PaystackApiError` if the API request fails

**Example:**

```php
$response = Paystack::api('/plan', 'get');
$plans = $response->json('data');
```

## Transaction Methods

### `charge()`

Charge a customer.

```php
public static function charge(array $data): Response
```

**Parameters:**

- `$data` (array): The charge data

**Example:**

```php
$response = Paystack::charge([
    'email' => 'customer@email.com',
    'amount' => 500000,
]);
```

### `makePaymentRequest()`

Initialize a transaction.

```php
public static function makePaymentRequest(array $data): Response
```

**Parameters:**

- `$data` (array): The transaction initialization data

**Example:**

```php
$response = Paystack::makePaymentRequest([
    'email' => 'customer@email.com',
    'amount' => 1000000,
]);

$authorizationUrl = $response->json('data.authorization_url');
```

### `chargeAuthorization()`

Charge an authorization.

```php
public static function chargeAuthorization(array $data): Response
```

**Parameters:**

- `$data` (array): The authorization charge data

**Example:**

```php
$response = Paystack::chargeAuthorization([
    'email' => 'customer@email.com',
    'amount' => 500000,
    'authorization_code' => 'AUTH_xxxxx',
]);
```

### `refund()`

Refund a transaction.

```php
public static function refund(array $data): Response
```

**Parameters:**

- `$data` (array): The refund data

**Example:**

```php
$response = Paystack::refund([
    'transaction' => 'TRX_xxxxx',
    'amount' => 500000,
]);
```

### `fetchTransactions()`

Get all the transactions that have happened overtime.

```php
public function fetchTransactions(): Response
```

**Example:**

```php
$response = Paystack::fetchTransactions();
$transactions = $response->json('data');
```

```php
$response = Paystack::checkAuthorization([
    'email' => 'customer@email.com',
```

**Parameters:**

## Plan Methods

### `createPlan()`

Create a plan.

```php
public static function createPlan(array $data): Response
```

**Parameters:**

- `$data` (array): The plan data

**Example:**

```php
$response = Paystack::createPlan([
    'name' => 'Premium Plan',
    'amount' => 500000,
    'interval' => 'monthly',
]);

$plan = $response->json('data');
```

### `fetchPlans()`

Get all the plans that you have on Paystack.

```php
public static function fetchPlans(): Response
```

**Example:**

```php
$response = Paystack::fetchPlans();
$plans = $response->json('data');

foreach ($plans as $plan) {
    echo $plan['name'];
    echo $plan['amount'];
}
```
